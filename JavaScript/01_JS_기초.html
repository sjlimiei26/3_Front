JS 기초

웹의 3요소

  • HTML : 웹 문서의 구조와 내용을 정의 (DOM 트리 구성)
  • CSS : DOM 요소의 스타일과 디자인을 편집
  • JavaScript : 이벤트가 발생했을 때 DOM을 동적으로 조작

자바스크립트란?

웹 브라우저에서 주로 사용되는 인터프리터 방식의 스크립트 언어
크롬 등의 브라우저에서 V8과 같은 자바스크립트 엔진이 내장되어 있어 별도의 설치 없이 실행 가능
* 인터프리터 방식 : 코드를 한 줄씩 읽어가며 즉시 실행하는 방식

자바스크립트 적용 방식

  1. 인라인 : HTML 태그의 이벤트 속성에 직접 자바스크립트 코드를 작성
  2. 내부 : HTML 문서 내에 script 태그 내에 자바스크립트 코드를 작성
  3. 외부 : 별도의 .js 파일을 생성하여 자바스크립트 코드를 작성

인라인 방식

<태그 on이벤트명="자바스크립트 코드">

내부 방식

script 태그 내에 자바스크립트 코드 작성
HTML 요소를 조작할 경우, 스크립트가 대상 요소보다 아래에 위치해야 정상적으로 인식됨

외부 방식

별도의 자바스크립트 파일(.js)을 생성하여 따로 관리하는 방식
