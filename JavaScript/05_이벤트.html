자바스크립트 이벤트

* 이벤트 : 브라우저에서 발생하는 사용자의 동작이나 상태 변화를 의미함 (click, keydown, keyup, drag, submit, load 등)
* 이벤트 핸들러 : 특정 이벤트가 발생했을 때 실행되는 콜백 함수

[1] 고전 이벤트 모델

element.onclick = 함수

동일한 이벤트에 하나의 핸들러만 등록 가능 (덮어씀). onclick = null 로 제거 가능.

표준 이벤트 모델

element.addEventListener(이벤트명, 핸들러)

동일한 이벤트에 여러 핸들러 등록 가능. removeEventListener() 로 특정 핸들러 제거 가능

target / currentTarget
